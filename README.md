# Expense Tracker

A web-based expense tracking application that allows users to record, manage, and review expenses efficiently.

## Features
- Add and manage expenses
- Categorize expenses
- Simple and clean UI
- Client-side validation

## Tech Stack
- HTML
- CSS
- JavaScript

## Project Structure
- `Project_Files/expense-tracker` – application source code
- `ExpenseTracker_Complete_Portfolio.pptx` – project presentation

## How to Run
1. Download or clone the repository
2. Open `index.html` in a browser
